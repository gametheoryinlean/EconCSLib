# Coalitional-Game Topic Catalog

Canonical folder topic: `game_theory.cooperative_game`

## Scope

Coalitional games with transferable utility, following MSZ Chapters 16-18.
This staged area covers the characteristic-function model, imputations,
the core, balancedness and the Bondareva-Shapley theorem, convex games,
and Shapley-value axioms and formulae.

The Lean backing currently lives in:

- `EconCSLib.GameTheory.CoalitionalGame.Basic`
- `EconCSLib.GameTheory.CoalitionalGame.Core`
- `EconCSLib.GameTheory.CoalitionalGame.ShapleyValue`

## References

- [MSZ Ch.16-Ch.18] Maschler, Solan, Zamir, *Game Theory*. Primary source for TU games, the core,
  balancedness, convex games, and the Shapley value.
