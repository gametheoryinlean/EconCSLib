# Belief System

A belief system assigns to each information set $Q_i$ a probability distribution
$\mu(Q_i)$ over the nodes of $Q_i$. Given beliefs and a behavioral profile, a
player can evaluate expected continuation payoff for each action available at the
information set.

- [MFoGT, Section 6.4.2] Laraki, Renault, and Sorin, *Mathematical Foundations of Game Theory*. A belief system assigns a probability distribution to each information set.
