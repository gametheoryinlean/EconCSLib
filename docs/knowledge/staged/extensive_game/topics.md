# Staged Extensive Game Topic Catalog

Canonical folder topic: `extensive_game`

## Subtopics

- `extensive_game.core` - arenas, histories, game trees, subgames, and induced outcomes.
- `extensive_game.perfect_information` - perfect-information strategies, backward induction, Zermelo-style determinacy, and finite-tree values.
- `extensive_game.imperfect_information` - information sets, pure strategies, perfect recall, and imperfect-information SPE targets.
- `extensive_game.normal_form` - normal-form and agent-normal-form reductions.
- `extensive_game.equilibrium` - Nash, subgame-perfect, behavioral, sequential, perfect, proper, and stable equilibrium concepts.
- `extensive_game.examples` - poker, entry, bargaining, chomp, and other example nodes.

## Boundary

Put normal-form equilibrium facts under `strategic_games` unless the statement
depends essentially on histories or subgames.
