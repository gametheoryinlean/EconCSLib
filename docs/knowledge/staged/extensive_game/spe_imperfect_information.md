# Subgame-Perfect Equilibrium In Imperfect-Information Games

In a finite extensive form game with perfect recall, a subgame-perfect equilibrium
is a behavioral strategy profile that prescribes a Nash equilibrium in every
subgame.

This is the imperfect-information analogue of the perfect-information definition,
using subroots to determine which continuations count as subgames.

## References

- [MFoGT, Def. 6.4.2] Laraki, Renault, and Sorin, *Mathematical Foundations of Game Theory*. SPE as a behavioral strategy profile prescribing a Nash equilibrium in every subgame.
