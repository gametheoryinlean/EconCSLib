# Market-Design Matching: One-to-One Topic Catalog

Canonical folder topic: `market_design.matching.one_to_one`

## Scope

One-to-one stable matching (marriage market) per Gale-Shapley 1962 and
Roth-Sotomayor 1990. Covers the core data structures (matching market,
matching, stability) and the four central theorems: Gale-Shapley
existence, proposing-side optimality, lattice structure of stable
matchings, and Roth 1982 strategy-proofness for the proposing side.

Many-to-one (college admission), housing markets (TTC), and assignment
games (Demange-Gale-Sotomayor) live under sibling subtopics
`market_design.matching.{many_to_one,housing,assignment}`.
