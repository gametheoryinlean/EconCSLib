# Brouwer Fixed Point Theorem

Let $C$ be a nonempty convex compact subset of a finite-dimensional Euclidean
space. Every continuous function
$$
  f:C\to C
$$
has a fixed point.

## References

- [MFoGT, Thm. 4.11.4] Laraki, Renault, and Sorin, *Mathematical Foundations of Game Theory*. Brouwer fixed point theorem for compact convex subsets of finite-dimensional Euclidean space.
