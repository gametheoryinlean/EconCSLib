# Common Knowledge Can Filter Nash Equilibria

MFoGT gives finite-game examples showing that Nash stability alone does not
require players to know opponents' payoff functions. When payoffs and rationality
are common knowledge, some Nash equilibria may be ruled out because a player can
anticipate another player's best-response behavior.

The same section also gives an example where common knowledge and pre-play
communication still do not remove an inefficient equilibrium without an additional
principle such as Pareto dominance.

## References

- [MFoGT, Section 4.10.4] Laraki, Renault, and Sorin, *Mathematical Foundations of Game Theory*. Examples showing how common knowledge of the game and rationality can refine predictions.
