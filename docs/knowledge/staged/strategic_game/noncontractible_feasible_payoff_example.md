# Noncontractible Feasible Payoff Example

MFoGT Exercise 4.12.3 defines a three-player finite game using the complex cube
roots of unity. Player 1's payoff is the real part of $abc$, player 2's payoff is the
imaginary part of $abc$, and player 3's payoff is identically zero.

The exercise asks to compute the payoff function of the mixed extension and deduce
that the feasible payoff set of the game is not contractible.

## References

- [MFoGT, Section 4.12, Exercise 3] Laraki, Renault, and Sorin, *Mathematical Foundations of Game Theory*. Three-player finite game whose feasible payoff set is not contractible.
