# Finite Nash Equilibrium Computation Examples

MFoGT Exercise 4.12.1 asks for all pure and mixed Nash equilibria in several
finite games: four two-player matrix games and one three-player game where the
third player chooses which matrix is used.

This node records the exercise as a source of finite Nash computation examples.
The individual equilibrium computations can be split into separate example nodes
if needed.

## References

- [MFoGT, Section 4.12, Exercise 1] Laraki, Renault, and Sorin, *Mathematical Foundations of Game Theory*. Exercise asking to compute pure and mixed Nash equilibria in several finite games.
