# Nash Payoffs Are Individually Rational

In a finite strategic game, the payoff vector of any Nash equilibrium belongs to
$R_1$, the set of feasible and individually rational payoffs.

## References

- [MFoGT, Section 4.10.1] Laraki, Renault, and Sorin, *Mathematical Foundations of Game Theory*. Any Nash equilibrium belongs to R_1.
